Nucleon number

We used to call the nucleon number the “mass number”.

Calculating nucleon number

The nucleon number of an atoms is the total number of protons and neutrons in the atoms.

\text{Nucleon number} = \text{Number of protons} + \text{Number of neutrons}

Notation

We usually represent the nucleon number with the letter A.

Why add the protons and neutrons?

We established that electrons have almost no mass. When calculating the mass (which is related to the nucleon number) of an atoms, we can ignore the electrons and just add up the protons and neutrons, which are much heavier and so contribute almost all of the mass of the atoms.

And the protons and neutrons have almost exactly the same mass, so we can just add the two.

Relative mass

It’s important to note that this calculated mass is the relative mass of the atoms, compared to a single proton. It isn’t the absolute mass.
