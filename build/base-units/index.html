Base units

There are seven base ‘SI’ units. They are:

  • meter (m) - for length
  • kilogram (kg) - for mass
  • second (s) - for time
  • ampere (A) - for electrical current
  • kelvin (K) - for temperature
  • mole (mol) - for amount of substance
  • candela (cd) - for light intensity
