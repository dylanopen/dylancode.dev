Solving simultaneous equations using matrices

Solve 3x+2y=9 and 4x+5y=5

  • let M=\pmatrix{3&2\\4&5}
  • \det M=(3)(5)-(2)(4)=7
  • M^{-1}=\frac17\pmatrix{5&-2\\-4&3}
  • \pmatrix{x\\y}=\frac17\pmatrix{5&-2\\-4&3}\pmatrix{9\\5}=\frac17\pmatrix{35\\-21}
  • \pmatrix{x\\y}=\pmatrix{5\\-3}
  • x=5
  • y=-3
