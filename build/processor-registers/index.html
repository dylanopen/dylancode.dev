Processor registers

There are lots of different registers in the processor, but the main 6 are the:

  • Program counter (PC)
  • Current instruction register (CIR)
  • Memory address register (MAR)
  • Memory buffer register (MBR)
  • Status register
  • Accumulator

Program Counter (PC)Holds the memory address of the next instruction to be fetched
Current Instruction Register (CIR)Holds the instruction currently being decoded and executed
Memory Address Register (MAR)Holds the address of a memory location to be read from or written to
Memory Buffer Register (MBR)Holds data being transferred to or from memory
Status RegisterKeeps track of the processor’s current state, e.g. overflow or zero result flags
AccumulatorUsed to store intermediate results of arithmetic and logic operations
