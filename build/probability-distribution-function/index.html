Probability distribution function

We can represent the probability of each possible value of a discrete random variable using something similar to a probability density function (which we use all the time for continuous variables, but can also use for discrete random variables).

We call it a probability distribution function.

Notation

The probability distribution function for a discrete random variable X is:

P(X=x)=f(x)

Where we can replace:

  • x (lowercase) with the value we are finding the probability of
  • f(x) with the probability distribution function.
