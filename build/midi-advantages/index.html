MIDI advantages

  • It’s a more compact representation of the audio:
    • Takes up less storage space
    • Uses less bandwidth
    • Can be read from disk or streamed more quickly
  • It’s very easy to edit notes after recording
    • Because you can digitally change the notes played after playing them
  • Easy to change instruments
  • No data is lost through sampling an analogue signal
    • With normal audio files, sampling reduces quality
    • There’s no analogue sampling, so the notes can be perfectly recreated
