Adding surds

We can only add surds that have the same surd part (the part under the root).

When adding surds with the same surd part, we simply add the coefficients (the numbers in front of the surd).

Example: simplify 3\sqrt{2}+5\sqrt{2}

  • =(3+5)\sqrt{2}
  • =8\sqrt{2}

Example: simplify 7\sqrt{3}-2\sqrt{3}

  • =(7-2)\sqrt{3}
  • =5\sqrt{3}
