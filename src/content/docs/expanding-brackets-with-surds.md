---
title: Expanding brackets with surds
---

When expanding brackets that contain surds, we use the same methods as we do with
normal brackets, (e.g. the grid method).

We just need to remember how to [multiply surds together](/multiplying-surds).

### Example: Expand $(\sqrt{2} + 3)(\sqrt{2} - 5)$

Draw a grid with two rows and two columns, since we have two terms in each set
of brackets:

| $\times$   | $\sqrt{2}$ | $-5$       |
|------------|------------|------------|
| $\sqrt{2}$ |            |            |
| $3$        |            |            |

Multiply the terms:

| $\times$   | $\sqrt{2}$       | $-5$            |
|------------|------------------|-----------------|
| $\sqrt{2}$ | $2$              | $-5\sqrt{2}$      |
| $3$        | $3\sqrt{2}$      | $-15$           |

Remember that when multiplying two of the same surds together (e.g
$\sqrt2\times\sqrt2$), the result is the number inside the surd (in this case,
$2$).

Add the terms in the middle of the grid together to get the final answer:
- $2 - 5\sqrt{2} + 3\sqrt{2} - 15$
- $-5\sqrt{2} + 3\sqrt{2} = -2\sqrt{2}$ - add the whole number parts as the
  surd parts are the same.
- **Answer**: $-13 - 2\sqrt{2}$

### Example: Expand $(2\sqrt{3} + 4)(\sqrt{3} - 1)$

Draw a grid with two rows and two columns, since we have two terms in each set
of brackets:

| $\times$     | $\sqrt{3}$     | $-1$          |
|--------------|----------------|---------------|
| $2\sqrt{3}$  |                |               |
| $4$          |                |               |

Multiply the terms:

| $\times$     | $\sqrt{3}$         | $-1$              |
|--------------|--------------------|-------------------|
| $2\sqrt{3}$  | $6$                | $-2\sqrt{3}$        |
| $4$          | $4\sqrt{3}$        | $-4$              |

Add the terms in the middle of the grid together to get the final answer:
- $6 - 2\sqrt{3} + 4\sqrt{3} - 4$
- $-2\sqrt{3} + 4\sqrt{3} = 2\sqrt{3}$ - add the whole number parts as the
  surd parts are the same.

**Answer**: $2 + 2\sqrt{3}$
