## Definition
For positive real number b:

$$\log_a(b) = c \Leftrightarrow a^c = b$$

## Identities
| Identites | Property |
|------|------------|
| Number.1 | $\log_a(1) = 0$ |
| Number.2 | $\log_a(a) = 1$ |
| Number.3 | $\log_a(a^x) = x$ |
| Number.4 | $a^{\log_a(x)} = x$ |

## Logarithm Rules
| Logarithm Rule Name | Property |
|------|------------|
| Product Rule | $\log_a(x\cdot y) = \log_a(x) + \log_a(y)$ |
| Quotient Rule | $\log_a(\frac{x}{y}) = \log_a(x) - \log_a(y)$ |
| Power Rule | $\log_a(x^y) = y \log_a(x)$ |
| Base Power Rule | $\log_{a^y}(x) = \frac{1}{y} \log_a(x)$ |
| Change Base Rule | $\log_{y}(x) = \frac{\log_a(x)}{\log_a(y)}$ |
