---
title: Common calculus results
---

| Expression      | Derivative              | Integral                     |
|-----------------|-------------------------|------------------------------|
| $e^{kx}$        | $ke^{kx}$               | $\frac{1}{k}e^{kx} + c$      |
| $\ln x$         | $\frac{1}{x}$           | N/A                          |
| $\frac1x$       | $-x^{-2}$               | $\ln\|x\|$                   |
| $\sin x$        | $\cos x$                | $-\cos x + c$                |
| $\cos x$        | $-\sin x$               | $\sin x + c$                 |
| $-\sin x$       | $-\cos x$               | $\cos x + c$                 |
| $-\cos x$       | $\sin x$                | $-\sin x + c$                |
