---
title: Random variable
---

> A **random variable** is a variable which depends entirely on some level of
> chance. It can change every time it is observed.

## Probability distribution

A random variable will have a *probability distribution* which describes the
likelihood of each possible value of the random variable, or potentially, if
it is [continuous](/continuous-variable), the likelihood of the random variable
taking on a value within a certain range.
