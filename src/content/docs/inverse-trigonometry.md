---
title: Inverse trigonometry
---

When we write $\sin^{-1}x$, for example, it is ambiguous. Do we mean:
- the principle value for which $\sin \theta=x$,
- or the inverse function of $\sin x$ ($\frac1{\sin x})?

We can be more clear using this terminology:

|              | Inverse function | Reciprocol function |
|--------------|------------------|---------------------|
| $\sin x$     | $arc\sin x$      | $cosec x$           |
| $\cos x$     | $arc\cos x$      | $sec x$             |
| $\tan x$     | $arc\tan x$      | $cot   x$           |
