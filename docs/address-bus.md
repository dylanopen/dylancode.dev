# Address bus

> The address bus is a [bus](/computer-bus) that carries the addresses of
> memory locations from the processor to the main memory, so the main memory
> knows where to read or write data.

The number of bits the address bus has determines how many memory locations the 
processor can access. For most modern computers, the address bus is 64 bits
wide, which means that the processor can access $2^{64}$ memory locations.  
Older computers had narrower address buses, like 32 bits, which meant they could
only access $2^{32}$ memory locations.
