# Exchange particle

An *exchange particle* is a [[particle]] which is used to transfer a [[force]] in a [[particle interaction]].

## Types of exchange particles

- [[photon]]s for the [[electromagnetic force]]
- [[gluon]]s for the [[strong nuclear force]]
- [[gravitron]]s for the [[gravitational force]]
- [[w plus boson]]s, [[w minus boson]]s and [[z boson]]s for the the [[weak nuclear force]]
