# Lepton

> A **lepton** is a particle which is affected by the [[weak nuclear force]].

## Types of lepton

- [[electron]]
- [[muon]]
- [[neutrino]]

| Question | Answer |

| What is a lepton? | Particle affected by the weak nuclear force. |
