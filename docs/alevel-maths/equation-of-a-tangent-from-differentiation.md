# Finding the equation of a tangent using differentiation

## Example: The derivative of a curve at point $(4, -2)$ is $5$ - find the equation of the tangent

- Gradient = $5$.
- X-offset = $4$
- Y-offset = $-2$
- $(y--2)=5(x-4)$
- $y+2=5x-20$
- $y=5x-22$

## Example: The derivative of a curve at point $(-3, 7)$ is $-2$ - find the equation of the tangent

- Gradient = $-2$.
- X-offset = $-3$
- Y-offset = $7$
- $(y-7)=-2(x--3)$
- $y-7=-2(x+3)$
- $y-7=-2x-6$
- $y=-2x+1$
