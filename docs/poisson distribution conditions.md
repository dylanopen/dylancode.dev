# Poisson distribution conditions

Just like how there are [[binomial distribution conditions]], there are also conditions for when we can use the [[poisson distribution]].

- The events are completely [[independent events|independent]] of each other.
- The events occur one at a time.
- The average number of events in a fixed interval of time or space is known ([[mean]] is constant).

## [[flashcards]]

| Question | Answer |
| --- | --- |
| The events must be completely independent of each other. | Poisson distribution condition |
| The events must occur one at a time. | Poisson distribution condition |
| The average number of events in a fixed interval of time or space (the mean) must be known and constant. | Poisson distribution condition |
