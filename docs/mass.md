# Mass

**Mass** is the fundamental unit for measuring *how much 'stuff' is in an object*.

In slightly more scientific terms, it is the *amount of matter in a substance*.

## Units

Mass is measured in [[kilogram]]s.

*The [[base units]] for mass is [[kilogram]]s, not [[gram]]s. It's the only measurement where the base unit starts with 'kilo'.*
