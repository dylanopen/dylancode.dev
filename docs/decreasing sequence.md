# Decreasing sequence

For any decreasing [[sequence]], it is **always** true that:

> $u_{n+1}<u_n$

Where:
- $u_n$ is a term in the sequence
- $u_{n+1} is the term in the sequence **after** $u_n$

## [[flashcards]]

| Question | Answer |
| --- | --- |
| What condition must be true for any decreasing sequence? | $u_{n+1} < u_n$ |
| What does $u_n$ represent in a sequence? | A term in the sequence |
| What does $u_{n+1}$ represent? | The term in the sequence after $u_n$ |
