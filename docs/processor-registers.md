# Processor registers

There are lots of different [registers](/register) in the [[processor]], but the main 6 are the:
- [Program counter](/program-counter) (PC)
- [Current instruction register](/current-instruction-register) (CIR)
- [Memory address register](/memory-address-register) (MAR)
- [Memory buffer register](/memory-buffer-register) (MBR)
- [Status register](/status-register)
- [Accumulator](/accumulator)
