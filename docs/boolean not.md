# Boolean NOT

To represent a [[not gate|NOT gate]] of `A` (in other words, invert the value of `A`), we
put a line on top of the [[expression]] we want to invert:

$$
\overline A
$$

Or, for a longer [[expression]] (inverting $A+B\cdot C$):

$$
\overline{A+B\cdot C}\oplus D
$$

## [[flashcards]]

| Question | Answer |
| --- | --- |
| Boolean NOT | To invert a value $A$, we use $\overline A$. |
| Boolean NOT for long expressions | How do you invert a longer expression like $A+B\cdot C$?	Use an overline over the whole expression, e.g. $\overline{A+B\cdot C}$. |
| Boolean NOT in complex expressions | What is the result of applying NOT to $A+B\cdot C$ then XOR with $D$?	$\overline{A+B\cdot C}\oplus D$ |
