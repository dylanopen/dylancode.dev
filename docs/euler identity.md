# Euler's identity

> Euler's identity states that:
> - $e^{i\pi}+1\equiv0$

If we substitute $\theta=\pi$ into [[euler formula]], we get Euler's *identity*.

- We're working in radians
- Euler's formula is $e^{i\theta}\equiv\cos\theta+i\sin\theta$
- Substitute $\theta$ for $\pi$:
    - $e^{i\pi}\equiv\cos\pi+i\sin\pi$
- Now, evaluate $\cos\pi$ and $\sin\pi$:
    - $e^{i\pi}\equiv-1+i(0)$
- We then have to rearrange / simplify to make it look fancier :\)
    - $e^{i\pi}\equiv-1$
    - $e^{i\pi}+1\equiv0$
