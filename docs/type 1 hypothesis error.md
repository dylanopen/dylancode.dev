# Type 1 hypothesis error

> In a *type one* hypothesis error, we have a [[false positive]] - we state that we have sufficient evidence to reject the [[null hypothesis]], but actually, the [[null hypothesis]] is still true.

## Significance level

The higher the [[significance level]], the **more likely** we are to have type 1 errors.

## Probability of a type 1 error

The chance of having a type 1 error is equal to the calculated [[significance level]] - that is, the significance level we calculate when carrying out the [[hypothesis test]] (not the thing we're testing for).

| Question | Answer |

| What is a Type 1 hypothesis error? | A *false positive* - we state we have sufficient evidence to reject the [[null hypothesis]], but it is actually still true. |

| What is the probability of a Type 1 error equal to? | The calculated [[significance level]] from the [[hypothesis test]] (not the pre-set threshold). |
