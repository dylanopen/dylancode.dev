# Continuous random variance

When we calculate the [[variance]] of a [[continuous-random-variable]], it's exactly the same as [calculating the variance](discrete-random-variance) of a [[discrete-random-variable]].

> $$
> Var(X)=E(X^2)-E(X)^2
> $$

To find $E(X^2)$, see [[continuous-random-square-expectation]].
