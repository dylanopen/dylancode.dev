# Density

Density measures the amount of [[mass]] per unit [[volume]].

## Equation

## $\text{density}=\frac{\text{mass}}{\text{volume}}$

> $\rho=\frac mV$

- Density is measured in [[kilogram]]s per [[metre]] cubed ($kg\,m^{-3}$)
- [[mass]] is measured in [[kilogram]]s ($kg$)
- [[volume]] is measured in [[metre]]s cubed ($m^3$)

## Units

- The SI unit for density is $kg\,m^{-3}$.
- We can also use $g\,cm^{-3}$.
