# Graph tree

A *tree* is a type of [[graph]] that has the [[properties]] of:
- [[connected graph|being connected]]
- having **no** [[graph loop|loops]]

This means that there is exactly one path between any pair of
[[graph vertex|vertices]] in the tree.

It also means that it is impossible to start at one vertex and get back to it,
**without** repeating a vertex and edge along the way.

## [[flashcards]]

| Question | Answer |
| --- | --- |
| A tree | Is a connected graph with no loops; there is exactly one path between any pair of vertices, and it is impossible to start and return to a vertex without repeating a vertex/edge. |
