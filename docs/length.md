# Length



## [[flashcards]]

| Question | Answer |
| --- | --- |
| Length | The measurement of something from end to end, or the longer dimension of an object. |
| Time | The ongoing sequence of events; duration from past to present to future. |
| How is length defined in physics? | Length is the distance between two points in space, measured in meters. |
| What is the SI base unit for length? | Meter (m) |
| What does the concept of space refer to? | Space is the boundless three-dimensional extent in which objects and events have relative position and direction. |
| How are length and space related? | Length is a measure of the distance between two points within space. |
| What does "measuring" mean? | Measuring is assigning a number to a property (like length) by comparing it to a standard unit. |
| What instruments are used to measure length? | Rulers, measuring tapes, calipers, micrometers, laser rangefinders. |
| Convert 1 meter to centimeters: | $1\, \text{m} = 100\, \text{cm}$ |
| Convert 1 kilometer to meters: | $1\, \text{km} = 1000\, \text{m}$ |
| How many millimeters in 1 meter? | $1\, \text{m} = 1000\, \text{mm}$ |
| What is $\log_5{125}$? | $\log_5{125}$ evaluates to 3 because $5^3 = 125$. |
| What does the $\log$ function represent? | The logarithm of a number is the exponent to which the base must be raised to produce that number. |
