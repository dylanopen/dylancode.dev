# Conic section

If we imagine a *cone* - a 3D shape that has a circular base and a pointed top - and we slice through it in lots of different directions, we can get some common shapes you may have seen before.

The shapes we get are called *conic sections*.

## Types of conic sections

The type of conic section we get depends on the angle at which we slice through the cone. We can get:
- A [[circle]]
- An [[ellipse]]
- A [[parabola]]
- Or a [[hyperbola]].
