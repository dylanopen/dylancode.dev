# Continuous random point distribution

> The [[point distribution]] of a [[continuous random variable]] at *any point* is $0$.

That's because the [[point distribution]] measures the probability of getting *exactly* that value. In continuous random [[cs/variables|variables]], there are infinitely many values, so it's very unlikely (impossible, in fact) that we will, say, measure the size of an apple tree to be **exactly** 2.71 metres.
