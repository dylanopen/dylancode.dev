# Memory address

A memory address is just the number we give to represent a location in [[main memory]]. It's like a label that we can use to refer to a specific [[byte]] or group of bytes in memory.

## [[flashcards]]

| Question | Answer |
| --- | --- |
| Memory address | The number used to represent a specific location in main memory, acting as a label to refer to a specific byte or group of bytes. |
