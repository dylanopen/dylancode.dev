| What is stratified sampling? | Dividing the population into subgroups (strata) and taking a proportional sample from each |
| What is quota sampling? | Interviewers select a specific number (quota) of individuals from each subgroup; not random |
| What is the median? | The middle value when data is arranged in order; for even $n$, it is the mean of the middle two |
| What is the mean? | The sum of all data values divided by the number of values: $\bar{x} = \frac{\sum x}{n}$ |
| What is the mode? | The value that appears most frequently in the dataset |
| What is the range? | The difference between the maximum and minimum values in the dataset |
| What is the interquartile range (IQR)? | The difference between the upper quartile ($Q_3$) and lower quartile ($Q_1$); IQR = $Q_3 - Q_1$ |
| What is variance? | The average of squared deviations from the mean: $\text{Var}(X) = E[(X - \mu)^2]$ |
| What is standard deviation? | The square root of the variance: $\sigma = \sqrt{\text{Var}(X)}$ |
| What are mutually exclusive events? | Events that cannot happen at the same time; $P(A \cap B) = 0$ |
| What is the addition rule for mutually exclusive events? | $P(A \cup B) = P(A) + P(B)$ |
| What are non-mutually exclusive events? | Events that can occur at the same time; $P(A \cap B) \neq 0$ |
| What is the addition rule for non-mutually exclusive events? | $P(A \cup B) = P(A) + P(B) - P(A \cap B)$ |
| What are independent events? | Events where the outcome of one does not affect the other; $P(A \cap B) = P(A) \times P(B)$ |
| What conditions must hold for a binomial distribution? | Fixed number $n$ of independent trials, each with same probability of success $p$, each trial outcome is success/failure |
| What is the notation for a binomial distribution? | $X \sim B(n, p)$ |
| State the binomial distribution formula. | $P(X = x) = \binom{n}{x} p^x (1-p)^{n-x}$ |
| What is the expectation of a binomial distribution $X \sim B(n, p)$? | $E(X) = np$ |
| What is the variance of a binomial distribution $X \sim B(n, p)$? | $\text{Var}(X) = np(1-p)$ |
| What is a random variable? | A variable whose value is determined by the outcome of a random event |
| What is a discrete variable? | A variable that can take on a finite or countably infinite number of distinct values |
| What is a discrete random variable? | A random variable that can take only discrete values |
| What is the expectation of a discrete random variable $X$? | $E(X) = \sum x_i p_i$ |
| What is the variance of a discrete random variable $X$? | $\text{Var}(X) = E(X^2) - [E(X)]^2 = \sum x_i^2 p_i - (\sum x_i p_i)^2$ |
| How do you find the mode of a discrete random variable? | The value of $X$ with the highest probability $p_i$ |
| How do you find the median of a discrete random variable? | The smallest value of $X$ such that the cumulative probability is at least 0.5 |
| How do you transform a discrete random variable? | If $Y = aX + b$, then $E(Y) = aE(X) + b$ and $\text{Var}(Y) = a^2 \text{Var}(X)$ |
| What is a discrete uniform distribution? | Each of $n$ equally likely outcomes has probability $1/n$ |
| What is a probability density function (PDF) for a continuous random variable? | A function $f(x)$ such that $P(a \le X \le b) = \int_a^b f(x) dx$; total area = 1 |
| How do you find the probability that a continuous random variable lies between two values? | $P(a < X < b) = \int_a^b f(x) dx$ |
| How do you find the median of a continuous random variable? | Solve $\int_{-\infty}^m f(x) dx = 0.5$ |
