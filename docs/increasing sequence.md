# Increasing sequence

For any increasing [[sequence]], it is **always** true that:

> $u_{n+1}>u_n$

Where:
- $u_n$ is a term in the sequence
- $u_{n+1} is the term in the sequence **after** $u_n$

## [[flashcards]]

| Question | Answer |
| --- | --- |
| What is always true for any increasing sequence? | $u_{n+1} > u_n$ |
| In the inequality $u_{n+1} > u_n$, what does $u_n$ represent? | A term in the sequence |
| In the inequality $u_{n+1} > u_n$, what does $u_{n+1}$ represent? | The term in the sequence after $u_n$ |
