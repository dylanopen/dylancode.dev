# Nuclide notation

When we represent the structure of a [[nuclide]], we need to show two things:
- Its [[proton-number]]
- Its [[nucleon-number]]

## Notation

We represent it in this form:

> $$
> ^A_ZEl
> $$

Where:
- $A$ is the [[nucleon-number]]
- $Z$ is the [[proton-number]]
- $El$ represents any symbol for an element (could be $C$, $Li$, $Og$, etc.)
