# Oscillating sequence



## [[flashcards]]

| Question | Answer |
| --- | --- |
| **Oscillating sequence** | An oscillating sequence is a sequence whose terms do not approach a single finite limit or diverge to $\pm\infty$; instead, the terms move back and forth between different values or ranges. |
| **How to identify an oscillating sequence?** | If a sequence does not converge and does not diverge to $+\infty$ or $-\infty$, it is oscillating. |
| **Example of oscillating sequence: $a_n = (-1)^n$** | The terms alternate between $1$ and $-1$, so the sequence oscillates. |
| **Example of oscillating sequence: $a_n = \sin(n)$** | The terms take values in $[-1, 1]$ but do not settle to a single limit, oscillating indefinitely. |
| **Can an oscillating sequence be bounded?** | Yes, many oscillating sequences (such as $(-1)^n$) are bounded but still oscillate. |
| **What is the difference between a convergent sequence and an oscillating sequence?** | A convergent sequence approaches a single finite limit; an oscillating sequence does not settle to any limit, often moving between values. |
| **What is a divergent sequence?** | A sequence that does not converge; includes both sequences that diverge to $\pm\infty$ and oscillating sequences. |
| **Example: $a_n = (-2)^n$** | This sequence oscillates in sign but its magnitude grows without bound; it does not converge but also does not oscillate in a bounded sense. |
