# Applications of differentiation

There are a few things we can do with the [[differentiation|derivative]] of a function, which are really useful for solving problems.

A reminder that use can use the [[power rule for differentiation]] or [[differentiation from first principles]] to calculate the derivative of a function.

Some things you can do with the derivative are:
- Finding the [[tangent from differentiation]] at a point
- Finding the [[normal from differentiation]] at a point
- Finding [[stationary points from derivative]]

## [[flashcards]]

| Question | Answer |
| --- | --- |
| applications of differentiation | There are a few things we can do with the derivative of a function, such as finding tangents, normals, and stationary points. |
| finding the tangent from differentiation | To find the tangent at a point, calculate the derivative and evaluate it at that point to get the slope $m$, then use the point-slope formula $y - y_1 = m(x - x_1)$. |
| finding the normal from differentiation | The normal is perpendicular to the tangent, so its slope is the negative reciprocal: $m_{\text{normal}} = -\frac{1}{m_{\text{tangent}}}$. |
| stationary points from derivative | Stationary points occur where the derivative $f'(x) = 0$. These can be local maxima, minima, or points of inflection. |
