# Assembler

- Assemblers convert low-level [[assembly]] language code into (even lower level) 
  [[machine code]].
- Assembly is specific to a particular [[instruction set architecture]], so the resulting machine code will only run on a [[processor]] which targets that specific ISA.

## [[flashcards]]

| Question | Answer |
| --- | --- |
| Assembler | Assemblers convert low-level assembly language code into machine code. |
| Assembly language specificity | Assembly is specific to a particular instruction set architecture (ISA), so the resulting machine code will only run on a processor which targets that specific ISA. |
| What is the purpose of an assembler? | To convert low-level assembly language into machine code. |
| What determines the processor that can run assembled code? | The instruction set architecture (ISA) that the assembly language targets; the resulting machine code runs only on processors supporting that ISA. |
