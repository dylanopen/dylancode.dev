# Logical topology

As opposed to a [[physical topology]], a **logical** [[network topology]] describes:

> How the data and [[network frame|frames]] flow around the network.

It's all about the architecture of the communication mechanism in the [[network]].

A [[network hub|hub]], for example, makes a [[physical topology|physical]] [[network switch|switch]] topology into a [[logical topology|logical]] [[bus topology|bus]] topology.
