# Plastic deformation

Plastic [[deformation]] is when an object's shape is permanently changed.

After the [[force]] ([[tension]] or [[compression]]) that deformed it is removed, it does not return to its original shape (it's permanently deformed).

| Question | Answer |
| --- | --- |
| What is plastic deformation? | Plastic deformation is when an object's shape is permanently changed after a force is removed. |

| What types of force cause plastic deformation? | Tension or compression. |
