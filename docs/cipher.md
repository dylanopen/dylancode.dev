# Cipher

A *cipher* is a particular function which we use to [encrypt](/encryption) and
[decrypt](/decryption) messages. It's the mathematical function that takes the
plaintext and produces the ciphertext, and the reverse cipher function does the
opposite - it takes the ciphertext and produces the plaintext.

## Types of cipher

There are thousands of different possible ciphers, but a few common ones are:
- [Caesar cipher](/caesar-cipher)
- [Vernam cipher](/vernam-cipher)
