# Bus width

The width of a [bus](computer-bus) is the number of bits it can transfer at a time. For example,
if a bus is 64 bits wide, it can transfer 64 bits of data at a time - usually
every [processor clock](/processor-clock) cycle.

## Performance impact

If the [address bus](address-bus) and [data bus](data-bus) are wider, the processor can access more memory and transfer more data at a time, which can improve performance. For example, a 64-bit data bus can transfer twice as much data as a 32-bit data bus in the same amount of time.
