# Applications of differentiation

There are a few things we can do with the derivative of a function, which are really useful for solving problems.

A reminder that use can use the [[power-rule-for-differentiation]] or [[differentiation-from-first-principles]] to calculate the derivative of a function.

Some things you can do with the derivative are:
- Finding the [[tangent-from-differentiation]] at a point
- Finding the [[normal-from-differentiation]] at a point
- Finding [[stationary-points-from-derivative]]
