# Energy conversions

We can represent *energy* with various units. The main ones being:
- Joules (J)
- Electronvolts (eV)
- Kilowatt-hours (kWh)

## Joules

$1J$ is equal to:
- $6.24\times10^{18}$ eV
- $2.78\times10^{-7}$ kWh

## Electronvolts

$1eV$ is equal to:
- $1.60\times10^{-19}$ J
- $4.45\times10^{-26}$ kWh

## Kilowatt-hours

$1kWh$ is equal to:
- $3.60\times10^{6}$ J
- $2.25\times10^{25}$ eV
