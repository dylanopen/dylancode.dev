# Simple graph

A simple graph is a type of [[graph]] which has no [[graph loop|loops]] or
multiple [[graph edge|edges]] between the same pair of
[[graph vertex|vertices]].

> **Simple graph**: a graph without [[loops]] or multiple edges between the same
> pair of vertices.
