# Logarithm graph equation

If we have an *exponential* graph with this equation:

$$
N=Ae^{kt}
$$

we can rewrite this as:

$$
\ln N = \ln A + kt
$$

We can then *plot* this data on a graph with a *logarithmic* scale on the
**y-axis**.
