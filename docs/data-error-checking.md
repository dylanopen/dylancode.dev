# Data error checking

When we send data from one place to another, we want to make sure that the data
is correct when it arrives. We don't want to send a message, and have it arrive
with some of the letters changed, or some of the numbers changed, because that
could cause all sorts of problems!

We can *verify* that the data is correct when it arrives by using *error
checking* - there are loads of different ways to do this.

## Common error checking methods

- [Parity bits](/parity-bit)
- [Majority voting](/majority-voting)
- [Checksums](/checksum)
