# Deductive sequence definition

We can define a [[sequence]] *deductively*.

This is otherwise known as finding a **position-to-term rule** for the sequence (i.e. $u_n=2n^2+3n-4$)

## [[flashcards]]

| Question | Answer |
| --- | --- |
| What is a deductive sequence definition? | Also known as a **position-to-term rule**, it defines a sequence by a formula relating the term value $u_n$ directly to its position $n$ (e.g. $u_n=2n^2+3n-4$). |
