# Inverse trigonometry

When we write $\sin^{-1}x$, for example, it is ambiguous. Do we [[mean]]:
- the principle value for which $\sin \theta=x$,
- or the inverse function of $\sin x$ ($\frac1{\sin x})?

We can be more clear using this terminology:

|              | Inverse function | Reciprocol function |
|--------------|------------------|---------------------|
| $\sin x$     | $[[arc]]\sin x$      | $cosec x$           |
| $\cos x$     | $[[arc]]\cos x$      | $sec x$             |
| $\tan x$     | $[[arc]]\tan x$      | $cot   x$           |

## [[flashcards]]

| Question | Answer |
| --- | --- |
| What is the ambiguity when writing $\sin^{-1}x$? | It could mean either the principle value for which $\sin \theta = x$ or the inverse function of $\sin x$ ($\frac{1}{\sin x}$). |
| What is the clearer terminology for the inverse function of $\sin x$? | $\arcsin x$ |
| What is the clearer terminology for the reciprocal function of $\sin x$? | $\csc x$ (cosec $x$) |
| What is the clearer terminology for the inverse function of $\cos x$? | $\arccos x$ |
| What is the reciprocal function of $\cos x$? | $\sec x$ |
| What is the clearer terminology for the inverse function of $\tan x$? | $\arctan x$ |
| What is the reciprocal function of $\tan x$? | $\cot x$ |
