# Stationary wave phase

In a [[stationary wave]], the particles between the [[wave node|nodes]] vibrate **in phase** with each other.

That means when one is going up, the other is going up, or when one is going down, the other is going down.

## [[flashcards]]

| Question | Answer |
| --- | --- |
| What is the phase relationship between particles between adjacent nodes in a stationary wave? | They vibrate **in phase** with each other. |
