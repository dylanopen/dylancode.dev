# Graph path

A **path** is just like a [[graph walk|walk]] on a [[graph]], except that
you are not allowed to traverse the same [[graph edge|edge]] **or**
[[graph vertex|vertex]] more than once.

## Visiting vertices and edges

- You **cannot** visit the same [[graph vertex|vertex]] multiple times in a
  path.
- You also **cannot** visit the same [[graph edge|edge]] multiple times in a
  path.
