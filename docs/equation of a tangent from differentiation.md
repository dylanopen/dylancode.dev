# Finding the equation of a tangent using differentiation

### Example: The derivative of a curve at point $(4, -2)$ is $5$ - find the equation of the tangent

- Gradient = $5$.
- X-offset = $4$
- Y-offset = $-2$
- $(y--2)=5(x-4)$
- $y+2=5x-20$
- $y=5x-22$

| How do you form the equation of a tangent given gradient $m$ and point $(x_1, y_1)$? | $y - y_1 = m(x - x_1)$ |
| What is the first step to find tangent equation when derivative at $(4, -2)$ is $5$? | Set gradient $m = 5$; point $(x_1, y_1) = (4, -2)$ |
| Write the equation of tangent using point-slope form for $m = 5$ at $(4, -2)$ | $y - (-2) = 5(x - 4)$ simplifies to $y + 2 = 5x - 20$ |
| After simplifying $y + 2 = 5x - 20$, what is the final tangent equation? | $y = 5x - 22$ |
| What do X-offset and Y-offset represent in tangent equation? | The coordinates of the point: X-offset = $x_1$, Y-offset = $y_1$ |
