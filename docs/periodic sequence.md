# Periodic sequence

A *periodic* [[sequence]] has a repeating sequence: the sequence *repeats* over the same terms forever.

We call the the length of the repeating terms the [[sequence period|period]] of the sequence.

## [[flashcards]]

| Question | Answer |
| --- | --- |
| Periodic sequence | Has a repeating pattern; the sequence repeats the same terms over and over. |
| Sequence period | The length of the repeating terms in a periodic sequence. |
