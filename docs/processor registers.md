# Processor registers

There are lots of different [[register|registers]] in the [[processor]], but the main 6 are the:
- [[program counter|Program counter]] (PC)
- [[current instruction register|Current instruction register]] (CIR)
- [[memory address register|Memory address register]] (MAR)
- [[memory buffer register|Memory buffer register]] (MBR)
- [[status register|Status register]]
- [[accumulator|Accumulator]]

## [[flashcards]]

| Question | Answer |
| --- | --- |
| Program Counter (PC) | Holds the memory address of the next instruction to be fetched |
| Current Instruction Register (CIR) | Holds the instruction currently being decoded and executed |
| Memory Address Register (MAR) | Holds the address of a memory location to be read from or written to |
| Memory Buffer Register (MBR) | Holds data being transferred to or from memory |
| Status Register | Keeps track of the processor's current state, e.g. overflow or zero result flags |
| Accumulator | Used to store intermediate results of arithmetic and logic operations |
