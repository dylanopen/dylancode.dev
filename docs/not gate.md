# NOT gate

A *NOT* [[logic gate]] takes a single input signal and produces an output signal that is
the **opposite** of the input signal.

## Truth table

The [[truth table]] for it looks like:

| Input A | Output (NOT A) |
|---------|----------------|
|    0    |      **1**     |
|    1    |      **0**     |
