# Continuous random variable

Unlike a [[discrete-random-variable]], a [continuous](continuous-variable), [random](random-variable) variable (CRV) can take on *any* value within a certain range - not just a set number. This means that instead of counting the number of outcomes, we [measure the probability](continuous-random-probability) of an outcome occurring within a specific interval.

The probability of getting *exactly* any specific value is actually zero for a CRV, because there are infinitely many possible values. either side of it.
